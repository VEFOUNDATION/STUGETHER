# STUGETHER
Gamified study app
STUGETHER

Gamified study app — an interactive prototype exploring what a social, gamified learning platform could look like, from onboarding through daily use.

This repo currently contains a single self-contained HTML prototype (INDEX.html) — no build step, no backend, no dependencies. Open the file in a browser, or enable GitHub Pages, to click through the flow.

What's in the prototype

The prototype covers the full arc of a user's experience, not just onboarding:

Onboarding — splash screen, intro carousel, sign-up/auth, role selection (student vs. course creator), university selection, avatar and "world" setup
Home & practice — a home dashboard, practice sessions, and session-complete/celebration screens
Courses — course browsing, a course hub, module pages with docs/video/questions, and a question bank
Creator tools — a course builder, creator home, creator profile, earnings, and a "go live" flow for course creators
Gamification — quests, a companion/leveling system, a reward store, and celebration moments
Community & social — a community feed, study rooms, and assignments
AI chat — a simulated study-assistant chat screen
Account & settings — profile, notifications, privacy, help, and withdrawal/payout screens (payout fields are placeholders — no real payment integration yet)
Status

This is a front-end-only, click-through prototype. Interactions like question generation from uploaded material, AI chat responses, and payouts are simulated/templated for demonstration purposes — they are not wired to a real backend or model. Building that integration is a planned next step, not part of this repo yet.

Running it

No install required:

Clone or download this repo
Open INDEX.html directly in a browser

Or view it live via GitHub Pages (if enabled for this repo).
